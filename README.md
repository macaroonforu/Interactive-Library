# Interactive-Library-
You can add a book. 


If you hit the check, book is marked as read. 


Hit the read button to change its status. 


Can remove books as well. 


CLICK THIS [LINK](https://macaroonforu.github.io/Interactive-Library-/) TO TRY IT OUT
